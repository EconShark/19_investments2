## 🎯 Objective
Sinh viên dựa trên code mẫu để xây dựng một **Financial Dashboard** phục vụ phân tích và theo dõi đầu tư, bao gồm:

- Phân tích **một tài sản tài chính** (cổ phiếu, trái phiếu, crypto, etc.)
- Xây dựng **dashboard danh mục đầu tư**
- Tích hợp **chatbot hỗ trợ phân tích tài chính**

---

## 📈 1. Single Asset Dashboard

### [1] Summary
- Hiển thị thông tin tổng quan của tài sản được chọn
- Danh sách tài sản có thể bao gồm:
  - Cổ phiếu (Việt Nam / Thế giới)
  - Trái phiếu
  - Bitcoin / Crypto
- Nội dung:
  - Giá hiện tại
  - Thay đổi (%)
  - Khối lượng giao dịch
  - Market cap (nếu có)
  - Các chỉ số cơ bản khác

---

### [2] Price & Volume Chart
- Hiển thị biến động giá theo thời gian
- Hỗ trợ lựa chọn:
  - Daily / Weekly / Monthly / Custom timeframe
- Loại biểu đồ:
  - Line chart
  - Candlestick chart
  - Volume bar chart

---

### [3] Statistical & Financial Analysis
Phân tích định lượng cho một tài sản:

- Returns:
  - Daily return
  - Log return
- Risk:
  - Volatility
  - Standard deviation
- Distribution:
  - Histogram returns
- Technical indicators (optional):
  - Moving Average (MA)
  - RSI
  - MACD

---

## 📊 2. Portfolio Analysis

### [4] Portfolio Analytics
Phân tích danh mục đầu tư:

- CAPM:
  - Beta
  - Expected return
  - Market risk vs idiosyncratic risk
- APT (Arbitrage Pricing Theory)
- Portfolio performance:
  - Sharpe ratio
  - Sortino ratio
  - Max drawdown
- Visualization:
  - Efficient frontier
  - Correlation heatmap
  - Asset allocation pie chart

---

### [5] Monte Carlo Simulation
- Mô phỏng phân phối giá trị danh mục trong tương lai
- Output:
  - Distribution of portfolio value
  - VaR (Value at Risk)
  - Expected shortfall
- Scenario simulation:
  - Best / worst case
  - Random walk price evolution

---

## 🤖 3. Chatbot Support System

### Features:
- Hỏi đáp về:
  - Giá cổ phiếu
  - Hiệu suất danh mục
  - Rủi ro đầu tư
  - Giải thích chỉ số tài chính
- Natural language interface
- Có thể tích hợp:
  - LLM API (OpenAI, etc.)
  - Rule-based fallback system

### Example queries:
- “Cổ phiếu này có rủi ro cao không?”
- “Danh mục của tôi có tối ưu không?”
- “Giải thích Sharpe ratio”
